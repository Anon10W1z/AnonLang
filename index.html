AnonLang

AnonLang is an extremely simplistic programming language written in Java.

Syntax

AnonLang uses extremely simplistic syntax. Let's jump right in!

var integerExample = 5;
var doubleExample = 3.14;
var stringExample = Hello;
//this is a comment
//let's re-assign some variables
integerExample = 3;
doubleExample = 3.1eh;
//the above will throw an IllegalAssignmentException for assigning a double a real value
//now let's evaluate some expressions
integerExample = &integerExample& * random();
doubleExample = sin(31);
//variable names are wrapped around with '&' to use them in expressions
//now let's loop!
repeat 10;
writeln My favorite number is &loopCounter&!;
//string concatenation can be done with "&conc&"
writeln Hello &conc&World!;
//lines are split by ';'
write Hello;writeln World;
//global variables can be split among programs
global var testGlobal = 3
writeln &C:\AnonTest.anon.testGlobal&
